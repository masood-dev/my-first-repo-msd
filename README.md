## Getting Started

## Common Git Commands

```bash
# Check status of your repository
git status

# Add files to staging area
git add <filename>

# Commit changes
git commit -m "Your commit message"

# Push changes to GitHub
git push

# Pull latest changes
git pull

# Create a new branch
git checkout -b <branch-name>

# Switch branches
git checkout <branch-name>
```
